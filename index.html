📖 Panduan Edit PhotoBooth

1. Cara Mengganti Tema

Buka file script.js, cari bagian "// DEFINISI TEMA"

// Contoh mengubah warna tema:
pastel: {
    bgPrimary: '#ffeef8',      // Ganti warna background
    accentColor: '#ff6b9d',   // Ganti warna aksen
    ...
}

2. Cara Menambah Tema Baru

// Di script.js, tambahkan tema baru:
namaTemaBaru: {
    bgPrimary: '#warna',
    accentColor: '#warna',
    // ... lanjutkan dengan variabel lain
},

// Lalu tambahkan tombol di index.html:
// <button onclick="changeTheme('namaTemaBaru')">Nama</button>

3. Cara Menambah Sticker

Tambahkan di bagian sticker-options di index.html:

<button onclick="selectSticker('🦄')">🦄</button>

4. Cara Mengganti Layout

Layout bisa 2, 3, 4, atau 6 foto. Untuk-edit CSS nya, cari di style.css bagian .photo-container
